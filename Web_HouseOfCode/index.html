Compétences

Les différentes compétences que vous allez acquérir durant cette formation.

Web Design

La création et la conception de site web ou web design est la conception de l'interface: l’organisation des pages, l’arborescence et la navigation dans un site web.

Développement Web

Un développeur web est un informaticien spécialisé dans la programmation ou des applications qui sont exécutées à partir d'un serveur web sur un navigateur web.

Revue de code

La revue de code (de l'anglais code review) est un examen systématique du code source d'un logiciel. Il permet d'analyser et de faire comprendre son code facilement.

Site Web adapté

Le Responsive est une approche de conception qui vise à l'élaboration de sites permettant une navigation pour l'utilisateur quelle que soit sa gamme d'appareil.

Graphisme

Vous allez également apprendre à utiliser des logiciels de graphismes durant votre formation. Ces derniers seront utiles à la création de maquette graphique.

Base de donnée

Une base de donnée permet de stocker et gérer l'ensemble des contenus générer sur un site web ou une application. Nécessaire à la réalisation d'un site dynamique.
